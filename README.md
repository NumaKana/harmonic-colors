# Harmonic Colors

コード進行を色彩とビジュアルエフェクトで表現する、音楽視覚化Webアプリケーション

## 概要

Harmonic Colorsは、和音進行を視覚的に体験できるツールです。調性を色相で、機能和声を色の変化で、テンションノートをきらめく粒子で表現し、共感覚的な音楽体験を提供します。

### 主な特徴

- **2色マーブル表現**: 調性（色1）と機能和声（色2）を同時に視覚化
- **カラースプレー**: テンションノートを美しい粒子エフェクトで表現
- **音声出力**: Web Audio APIによるリアルタイム音声生成
- **作曲支援**: キーとコード進行を入力し、視覚と聴覚で確認

- **TypeScript** - 型安全な開発
- **React** - UIコンポーネント

- **Tone.js** - 音楽的なWeb Audio API操作
- **Vite** - 高速な開発環境
